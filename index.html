Ser Tag Description
1 a Defines a hyperlink.
2 b Defines bold paragraph
3 em Defines italic
4 body Defines the document body
5 br Produces a single line break.
6 div Specifies a division or a section in a document.
7 h1-h6 Defines HTML headings.
8 html Defines the root of an HTML document.
9 footer Represents the footer of a document or a section.
10 nav Defines a section of navigation links.
11 p Defines a paragraph.
12 section Defines a section of a document, such as header, footer etc.
13 span Defines an inline styleless section in a document.
14 link Defines the relationship between the current document and an external resource.
15 meta Provides structured metadata about the document content.
